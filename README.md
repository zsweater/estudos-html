# 📚 Estudos de HTML5 - Página Informativa (Web Novel chinesa)

Este é o meu primeiro projeto prático no mundo do desenvolvimento web! Desenvolvi uma página informativa sobre a light novel **Demonic Emperor**, focando na aplicação dos conceitos fundamentais do HTML5.

## 🚀 Tecnologias Utilizadas
* **HTML5:** Estruturação de tags semânticas, títulos, parágrafos, listas e hiperlinks.
* **VS Code:** Editor de código.
* **Live Server:** Extensão para testes em ambiente local.
* **GitHub Pages:** Para hospedagem e publicação do site na internet.

## 🎯 Objetivos do Projeto
* Praticar a estrutura básica de um documento HTML (`<head>`, `<body>`, etc).
* Organizar conteúdo textual usando cabeçalhos (`<h1>` a `<h3>`) e parágrafos (`<p>`).
* Trabalhar com mídias inserindo imagens (`<img>`) organizadas em diretórios.
* Implementar links externos (`<a>`) direcionando para fontes de leitura.
